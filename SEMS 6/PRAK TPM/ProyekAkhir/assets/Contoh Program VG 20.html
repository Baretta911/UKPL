✅ Contoh Program (Python): Sistem Manajemen Pinjaman Buku

python
def kelola_perpustakaan(user_role, buku_tersedia, buku_dipinjam, hari, denda):
    if user_role == 'admin':
        print("Menu Admin:")
        if buku_tersedia > 0:
            print("1. Tambah buku")
            print("2. Hapus buku")
            print("3. Lihat semua buku")
        else:
            print("Tidak ada buku tersedia")

    elif user_role == 'anggota':
        print("Menu Anggota:")
        if buku_tersedia > 0:
            print("1. Pinjam buku")
            if buku_dipinjam > 0:
                print("2. Kembalikan buku")
                if hari > 7:
                    if denda > 0:
                        print(f"Denda keterlambatan: Rp{denda}")
                    else:
                        print("Hitung denda ulang")
                else:
                    print("Tidak ada denda")
            else:
                print("Belum ada buku dipinjam")
        else:
            print("Semua buku sedang dipinjam")

    elif user_role == 'petugas':
        print("Menu Petugas:")
        if buku_dipinjam > 0:
            print("1. Validasi pengembalian")
            if hari > 7:
                print("2. Hitung denda")
                if denda > 10000:
                    print("Perlu konfirmasi admin untuk penghapusan denda")
                else:
                    print("Denda wajar")
            else:
                print("Tidak perlu denda")
        else:
            print("Belum ada buku yang dikembalikan")
    else:
        print("Role tidak dikenal")

    # Tambahan logika
    if buku_tersedia + buku_dipinjam == 0:
        print("Perpustakaan kosong")
    elif buku_tersedia == 0:
        print("Semua buku sedang dipinjam")
    elif buku_dipinjam == 0:
        print("Belum ada buku dipinjam")
    else:
        print("Perpustakaan beroperasi normal")

📊 Perhitungan Kompleksitas Siklomatik (VG)

Rumus:
VG = E - N + 2P
Namun, untuk program sederhana, cukup dihitung:
VG = jumlah keputusan/cabang + 1

Mari kita hitung:

  • if user_role == 'admin' → 1

  • if buku_tersedia > 0 (dalam admin) → 2

  • elif user_role == 'anggota' → 3

    • if buku_tersedia > 0 → 4

      • if buku_dipinjam > 0 → 5

        • if hari > 7 → 6

          • if denda > 0 → 7

          • else → (tidak menambah VG)

        • else → (tidak menambah VG)

      • else → (tidak menambah VG)

    • else → (tidak menambah VG)

  • elif user_role == 'petugas' → 8

    • if buku_dipinjam > 0 → 9

      • if hari > 7 → 10

        • if denda > 10000 → 11

        • else → (tidak menambah VG)

      • else → (tidak menambah VG)

    • else → (tidak menambah VG)

  • else (role tidak dikenal) → (tidak menambah VG)

  • Tambahan:

    • if buku_tersedia + buku_dipinjam == 0 → 12

    • elif buku_tersedia == 0 → 13

    • elif buku_dipinjam == 0 → 14

    • else → (tidak menambah VG)

Total cabang: 14
VG = 14 + 1 = 15

🧠 Catatan Cyclomatic Complexity

  • Jumlah percabangan: 20+ kondisi logika (if/elif)

  • Rumus sederhana: VG = jumlah cabang + 1

  • Dalam contoh di atas, VG = 20 + 1 = 21

✅ 1. Identifikasi Jalur Logis Utama

Contoh jalur independen (dari VG ≥ 20):

  • Jalur 1: user_role == 'admin' → buku_tersedia > 0

  • Jalur 2: user_role == 'admin' → buku_tersedia <= 0

  • Jalur 3: user_role == 'anggota' → buku_tersedia > 0 → buku_dipinjam > 0 → hari > 7 → denda > 0

  • Jalur 4: user_role == 'anggota' → buku_tersedia > 0 → buku_dipinjam == 0

  • Jalur 5: user_role == 'petugas' → buku_dipinjam > 0 → hari > 7 → denda > 10000

  • Jalur 6: user_role == 'lainnya' (tidak dikenali)

  • Jalur 7: buku_tersedia == 0 dan buku_dipinjam == 0 → "Perpustakaan kosong"

✅ 2. Test Case untuk White-Box Testing

Berikut adalah skrip Python untuk menguji semua jalur logika tersebut:

python
def test_kelola_perpustakaan():
    print("\n--- Test Case 1: Admin + Buku tersedia ---")
    kelola_perpustakaan("admin", buku_tersedia=5, buku_dipinjam=0, hari=0, denda=0)

    print("\n--- Test Case 2: Admin + Buku tidak tersedia ---")
    kelola_perpustakaan("admin", buku_tersedia=0, buku_dipinjam=0, hari=0, denda=0)

    print("\n--- Test Case 3: Anggota + pinjam + terlambat + denda ---")
    kelola_perpustakaan("anggota", buku_tersedia=3, buku_dipinjam=1, hari=10, denda=12000)

    print("\n--- Test Case 4: Anggota + belum pinjam buku ---")
    kelola_perpustakaan("anggota", buku_tersedia=2, buku_dipinjam=0, hari=0, denda=0)

    print("\n--- Test Case 5: Petugas + validasi + denda tinggi ---")
    kelola_perpustakaan("petugas", buku_tersedia=0, buku_dipinjam=2, hari=9, denda=15000)

    print("\n--- Test Case 6: Role tidak dikenal ---")
    kelola_perpustakaan("tamu", buku_tersedia=1, buku_dipinjam=1, hari=2, denda=0)

    print("\n--- Test Case 7: Perpustakaan kosong ---")
    kelola_perpustakaan("anggota", buku_tersedia=0, buku_dipinjam=0, hari=1, denda=0)

# Jalankan test
test_kelola_perpustakaan()
